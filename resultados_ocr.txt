Aula5-Ambientes_Virtuais.png
Publicado em Data Science / Data Science por Millena Gená Pereira
' data science, Python tempo de vídeo: 10min

Ambientes virtuais são utilizados para separar projetos Python com suas dependências e
bibliotecas em um único lugar. O ambiente virtual é específico para o projeto e não interfere
nas dependências e especificações de outros projetos. Isso significa que podemos ter
diferentes ambientes virtuais na nossa máquina e cada um deles será criado especificamente

para um projeto.

Para criar e conseguir trabalhar com um ambiente virtual, é preciso conhecer alguns

comandos. São eles:

e Criarum ambiente: conda create --name nome do ambiente python=:
e Listar os ambientes existentes na máquina: conda env list

* Ativaro ambiente: conda activate nome do ambiente

e Verificar as bibliotecas instaladas no ambiente: conda list

* Instalar uma nova biblioteca no ambiente: conda install nome da biblioteca
* Desativar o ambiente: conda deactivate

* Deletar um ambiente: conda env remove --name nome do ambiente
